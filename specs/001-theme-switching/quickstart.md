# Quickstart: Theme Switching

## Using the Theme Switcher

1.  **Locate the theme switcher control**: The theme switcher is a dropdown menu located in the application's header.
2.  **Select a theme**: Open the dropdown to see the available themes: "Light", "Dark", and "System".
3.  **Apply the theme**: Click on your desired theme. The application's appearance will update instantly.

-   **"Light"**: Sets the application to a light color scheme.
-   **"Dark"**: Sets the application to a dark color scheme.
-   **"System"**: The application will automatically sync with your operating system's theme.
